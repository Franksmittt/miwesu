================================================================================
1. PROJECT OVERVIEW
================================================================================

MIWESU GAME FARM is a luxury private residence collection website located in the
Makoppa district, Thabazimbi, Limpopo, South Africa. The website showcases an
exclusive game farm offering luxury accommodations, ethical hunting experiences,
and conservation activities in the Arid Sweet Bushveld biome.

The website is built with Next.js 14 App Router, utilizing React Server Components
for optimal performance and SEO. The design philosophy emphasizes luxury, elegance,
and the natural beauty of the African bushveld.
